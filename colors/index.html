Colors and Stroke

Set Fill and Stroke colors before drawing a shape, path or text. Each color argument values must be float 0 and 1.

Chitra library provides four different ways to set the fill or stroke color. Default fill and stroke color is Black and default opacity is 1 (100% opaque).

Gray scale

0 is Black and 1 is White. In between values are gray values.

fill 0.85
stroke 0
rect 100, 100, 500, 500

OR

ctx.fill 0.85
ctx.stroke 0
ctx.rect 100, 100, 500, 500

Gray scale with Opacity

alpha 0 is full transparent and alpha 1 is Opaque.

Example: Black with 50% opacity

fill 0, 0.5
stroke 0
rect 100, 100, 500, 500

OR

ctx.fill 0, 0.5
ctx.stroke 0
ctx.rect 100, 100, 500, 500

RGB color

fill 1, 0, 0
stroke 0
rect 100, 100, 500, 500

OR

ctx.fill 1, 0, 0
ctx.stroke 0
ctx.rect 100, 100, 500, 500

RGBA color

fill 1, 0, 0, 0.5
stroke 0
rect 100, 100, 500, 500

OR

ctx.fill 1, 0, 0, 0.5
ctx.stroke 0
ctx.rect 100, 100, 500, 500

Using Hex Colors

background "#715842"

fill "#333333"
stroke "#daa520"
stroke_width 5

rect 100, 100, 100, 100

# With 50% Opacity
fill "#daa52080"
rect 125, 125, 50, 50

Fill and Stroke Opacity

fill "#555555"
stroke "#555555"
(0...10).each do |x|
  so = x > 10 ? 10 : x+1
  fill_opacity x/10
  stroke_opacity so/10

  rect x*30, 0, 30, 30
end

Disable fill

no_fill
stroke 0
rect 100, 100, 500, 500

OR

ctx.no_fill
ctx.stroke 0
ctx.rect 100, 100, 500, 500

Stroke width

Default Stroke width is 1. Set Stroke width as

stroke_width 10
rect 100, 100, 500, 500

OR

ctx.stroke_width 10
ctx.rect 100, 100, 500, 500

Disable Stroke

Use stroke_width 0 or no_stroke.

fill 0, 0, 1, 0.5
stroke_width 0
rect 100, 100, 500, 500
# OR use no_stroke
fill 0, 0, 1, 0.5
no_stroke
rect 100, 100, 500, 500

OR

ctx.fill 0, 0, 1, 0.5
ctx.stroke_width 0
ctx.rect 100, 100, 500, 500
# OR use no_stroke
ctx.fill 0, 0, 1, 0.5
ctx.no_stroke
ctx.rect 100, 100, 500, 500
